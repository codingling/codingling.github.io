Linux常用命令（测试开发）

https://www.cnblogs.com/Javame/p/3968343.html

1、初级常用命令整理

vim 文件名 编辑文件

cat 文件名 显示文件内容

chmod 664 文件名 修改文件权限为 -rw- rw- r–

ls la 查看当前目录下的所有内容包含隐藏的内容

ls l 查看当前目录下的内容的详细信息

ls 查看当前目录下的内容

touch 文件名 创建文件

tar 文件

source 文件名 在当前bash环境下读取并执行文件中的命令

pwd 显示当前目录

sync 在执行reboot或者shutdown等命令时一定要首先使用sync将内存中的信息同步到磁盘上，以免造成数据丢失

su 用户名 切换到某个用户

2、进程与通信

ps 查看当前进程

sleep 几秒 使当前状态延迟几秒

kill 信号 进程id 发送信号给某个进程使其状态发生改变

netstat 查看网络状态

ping ip地址 使用icmp协议查看该ip地址是否能联通

route print 查看路由表的全部内容

ifconfig 网卡信息
