使用Django框架的forum开发的知识点总结

2019-03-03
  • django

1、django工程目录结构优化待读：
http://www.loonapp.com/blog/11/
2、0.0.0.0代表网卡上所有的ip地址
3、ctrl + Tab切换浏览器tab
alt + Tab切换当前任务
ctrl + T新建一个窗口
ctrl + /注释
4、启动服务器：python manage.py runserver 0.0.0.0:8000
$ python manage.py migrate # 创建表结构
$ python manage.py makemigrations TestModel # 让 Django 知道我们在我们的模型有一些变更
$ python manage.py migrate TestModel # 创建表结构
5、为防止密码泄露，可使用变量代替密码，在本地配置环境变量，设置变量的值
6、git reset –hard FETCH_HEAD返回到上一次pull成功的状态
7、jason为数据格式，取代xml的存在，为轻量级的数据交换格式
ajax为快速创建动态网页、实现页面和web服务器之间数据的异步传输的技术
JQuery是一个兼容多个浏览器的javascript框架
8、在python中is判断地址是否相等，==判断值是否相等；在java中刚好与之相反
9、url路由问题，django当系统功能多样时，应使用分级路由
10、由于http协议本身是无状态的，cookie和session都可以记录客户端的状态
cookie是一种数据存储技术，将一段文本保存在客户端，cookie存在不可靠和不安全的特性
session是在服务器存储少量的信息，客户端发送请求时，在HTTP的请求头中，包含一行cookie信息。
服务器根据cookie信息得到session id，再根据session id找到对应的session，因此即使用户关闭了浏览器，
session仍然将保持到会话过期

12、http://www.cnblogs.com/yanglang/p/6697657.html表格
