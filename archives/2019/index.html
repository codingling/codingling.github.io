git使用

2019-06-01

在github上再次新建一个仓库

1、在本地建文件夹，并在该文件夹中执行git init，让git认识该文件夹

2、执行之后再执行 git remote add origin https://github.com/illovey/Defense-Management-System.git和远程仓库关联

3、接下来可以使用git add -A从工作区添加到暂存区、git commit将暂存区的提交到本地版本库、git push将本地版本库的分支推送到远程服务器上对应的分支

用git hexo管理博客

1、进入博客所在的文件夹

执行hexo new “博客的名字”，创建一篇博客

2、写好后执行hexo g -d，生成html，并将博客推送到远程仓库，实现浏览器可访问

毕业论文过程学习

2019-04-30

Assertions：[中]预言。检查应用状态，确认网页各状态是否与预期一致，如“确认网页的标题是XXX”、”确认checkbox is checked”。Assertions有三类：

  • assertXXX，预言失败，测试终止。
  • verifyXXX，verify失败，测试继续，日志记录，需要自己讲结果写入，case量特别大的时候建议使用这个，然后通过分析日志来检查执行结果。
  • waitForXXX，等待一些条件变为真，测试Ajax应用时特别有效。如果条件已经满足，不会等待，直接通过，如果失败，中断测试。等待时间由setTimeout来设置，等待时间超时，测试中断。

详细参见：https://www.jianshu.com/p/c5659b57fdf0

2019-03-22

划龙舟项目

需求分析：

1、控制龙舟的移动，躲避石头和拾取粽子

2、撞上石头减分，拾到粽子加分，随着分数增加，游戏难度加大

3、得分大于100时，游戏胜利，得分小于0时，游戏失败

4、要可以显示开发者信息，游戏背景的滚动，以及背景音乐和音效的控制

项目背景：

1、学了一些概念理论性的知识，如四大组件，布局，组件，以及线程的概念

2、实践过一些比较小型和琐碎的东西，比如说activity的跳转，多种布局，组件的实现，以及响应事件的实现

3、对于动态的背景滚动以及分数的持续计算等逻辑控制，背景音乐和音效的控制基本不懂

UI和UE设计：

1、使用PS设计游戏首页，首页中点击按钮查看开发者信息、游戏规则、以及activity的跳转

2、使用PS设计游戏页，游戏页上部分导航栏实现背景音乐和音效的开关

架构设计

1、用到的技术：activity，service实现背景音乐的播放，布局，surfaceview绘制机制lockcanvas -绘制-unlockcanvas保证线程安全，按钮，弹出菜单组件实现信息的展示和开关，开启子线程来负责绘制，soundpool实现音效的播放，handler实现监听分数并进行相应的处理

字节跳动春招题之校正字符串

2019-03-22

输入一串字母，如果存在AAA型的字符串则改为AA型，如果存在AABB型的字符串则去掉第二对中的第一个字符变为AAB，举个栗子helllo修改为hello，wooooooc修改为wooc，aaabb应修改为aab

解题思路：

由于考虑到删除次数较多，因此这里使用链表来存储字符串，判断是否存在相应类型的字符串，如果存在执行相应删除操作
